# sao-sample

> my kryptonian SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao sao-tina-template my-project
```

### From git

```bash
sao strange-fish/sao-tina-template my-project
```
